# BMI Calculator using Python

This is a simple command-line **BMI (Body Mass Index) Calculator** project developed using **Python**.  
The program takes user input for weight and height, calculates BMI, and displays the health category.

## üìå What is BMI?

BMI (Body Mass Index) is a value derived from the mass (weight) and height of a person.

### Formula:
BMI = weight (kg) / height (m)¬≤

##  Features

- User-friendly command-line interface
- Takes weight and height as input
- Input validation (no negative or invalid values)
- Calculates BMI accurately
- Displays BMI category:
  - Underweight
  - Normal Weight
  - Overweight
  - Obesity
- Option to calculate BMI multiple times
- Beginner-friendly and well-structured code
